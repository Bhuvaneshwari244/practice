## Quick Translation Options

### Option 1: Use Google Translate API (Recommended)
```bash
npm install @vitalets/google-translate-api
node translate-script.js
```

### Option 2: Manual Translation
Add translations following the same pattern in `src/data/translations.ts`

### Option 3: Keep English Fallback
The app works fine with English fallback for less common languages.
